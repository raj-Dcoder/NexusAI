# NexusAI

AI-powered enterprise knowledge assistant built using:
- FastAPI
- LangChain
- ChromaDB
- Sentence Transformers
- Gemini API

## Features
- PDF upload
- Text extraction
- Chunking
- Embeddings
- Vector database foundation

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload --port 8001
